# 文案风格生成器

基于视频文案风格生成新主题文案的网页应用。

## 功能特点

- **风格提取**：输入一个或多个视频文案，AI自动分析并提取文案风格特征
- **风格库管理**：保存和管理多个文案风格，随时复用
- **框架生成**：根据选定风格和新主题，生成3个不同角度的解读思路与文案框架
- **文案创作**：基于框架生成完整的视频文案，支持编辑和复制

## 技术栈

- **前端**：Next.js 14 + React 18 + TypeScript + Tailwind CSS
- **状态管理**：Zustand
- **AI模型**：阿里云千问 (qwen-plus)
- **部署**：Vercel

## 本地开发

```bash
# 安装依赖
npm install

# 配置环境变量
cp .env.example .env.local
# 编辑 .env.local，填入你的 API Key

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000 查看应用。

## 环境变量

| 变量名 | 说明 |
|--------|------|
| DASHSCOPE_API_KEY | 阿里云千问 API Key |

## 部署到 Vercel

1. Fork 本仓库
2. 在 Vercel 导入项目
3. 设置环境变量 `DASHSCOPE_API_KEY`
4. 部署完成

## 使用流程

1. **首页**：输入视频文案，提取风格并保存
2. **风格库**：选择一个风格，输入新主题
3. **框架生成**：AI生成3个备选框架，可编辑修改
4. **最终文案**：选择框架后生成完整文案
